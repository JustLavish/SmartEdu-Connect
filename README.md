# 🎓 SmartEdu Connect

## SDG 4 – Quality Education

SmartEdu Connect is a modern educational web application developed to support the United Nations Sustainable Development Goal 4 (Quality Education). The platform helps students discover learning resources, monitor their learning progress, and access educational content through an easy-to-use interface.

## 📌 Project Objective

The objective of this project is to provide students with a centralized platform where they can access educational resources, track their progress, and improve their learning experience.

## 🚀 Features

* Student Registration
* User Login System
* Resource Management
* Search Functionality
* Learning Progress Tracker
* Statistics Dashboard
* Featured Courses Section
* Feedback Form
* Dark Mode Support
* Responsive Bootstrap Design
* LocalStorage Data Persistence

## 🛠 Technology Stack

* HTML5
* CSS3
* JavaScript
* Bootstrap 5
* Font Awesome
* Google Fonts (Poppins)
* LocalStorage

## 🎯 Selected Sustainable Development Goal

**SDG 4 – Quality Education**

This project contributes to SDG 4 by providing students with easy access to learning resources and educational tools that promote lifelong learning opportunities.
